# Spider Portfolio

Vite + React 19 portfolio site using GSAP, Three.js, and Tailwind CSS v4.

## Setup
- Dev server: `npm run dev` (Vite on 0.0.0.0:5000, all hosts allowed for Replit proxy)
- Build: `npm run build` → `dist/`
- Deployment: static site, serves `dist/`

## Workflow
- `Start application` → `npm run dev` (port 5000, webview)
